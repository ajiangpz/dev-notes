## 题目

假设你正在爬楼梯。需要 `n` 阶你才能到达楼顶。

每次你可以爬 `1` 或 `2` 个台阶。你有多少种不同的方法可以爬到楼顶呢？

```
输入：n = 2
输出：2
解释：有两种方法可以爬到楼顶。
1. 1 阶 + 1 阶
2. 2 阶

输入：n = 3
输出：3
解释：有三种方法可以爬到楼顶。
1. 1 阶 + 1 阶 + 1 阶
2. 1 阶 + 2 阶
3. 2 阶 + 1 阶




```

## 解题思路

1. **基本情况**：
   - 当 `n = 1` 时，只有1种方法（爬1步）
   - 当 `n = 2` 时，有2种方法（爬1步+1步 或 直接爬2步）

2. **动态规划数组**：
   - 创建一个数组 `dp`，`dp[i]` 表示爬到第 `i` 阶楼梯的方法数
   - 初始化 `dp[1] = 1` 和 `dp[2] = 2`

3. **状态转移方程**：
   - 对于每个 `i` 阶楼梯，爬到它的方法数是爬到 `i-1` 阶和 `i-2` 阶的方法数之和
   - 公式：`dp[i] = dp[i-1] + dp[i-2]`

4. **遍历计算**：
   - 从 `3` 阶开始，遍历计算每个阶梯的方法数
   - 最终返回 `dp[n]` 即为爬到第 `n` 阶楼梯的方法数





## 代码实现

```js
var climbStairs = function(n) {
    if (n === 1) return 1;
    if (n === 2) return 2;
    let dp = new Array(n + 1);
    dp[1] = 1;
    dp[2] = 2;  
    for (let i = 3; i <= n; i++) {
        dp[i] = dp[i - 1] + dp[i - 2];
    }
    return dp[n];
};          

```
