## 题目描述

找出所有相加之和为 `n` 的 `k` 个数的组合，且满足下列条件：

- 只使用数字 `1` 到 `9`
- 每个数字 最多使用一次
- 返回所有可能的有效组合的列表。该列表不能包含相同的组合两次，组合可以以任何顺序返回。

## 示例

```
输入: k = 3, n = 7
输出: [[1,2,4]]
```

```
输入: k = 3, n = 9
输出: [[1,2,6], [1,3,5], [2,3,4]]
```

## 解题思路

1. 使用回溯算法
2. 从 `1` 到 `9` 中选择 `k` 个数，使得它们的和为 `n`
3. 使用递归实现回溯
4. 使用一个数组 `path` 存储当前路径
5. 使用一个变量 `start` 存储当前选择的起始位置
6. 使用一个变量 `result` 存储最终结果

## 代码实现

```js
var combinationSum3 = function (k, n) {
  const result = [];
  const dfs = (start, path) => {
    if (path.length === k) {
      if (path.reduce((a, b) => a + b, 0) === n) {
        result.push([...path]);
      }
      return;
    }
    for (let i = start; i <= 9; i++) {
      path.push(i);
      dfs(i + 1, path);
      path.pop();
    }
  };
  dfs(1, []);
  return result;
};
```
