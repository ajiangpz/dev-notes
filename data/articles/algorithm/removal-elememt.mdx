# 思路
左右双指针遍历，如果left的值等于目标值，则和right的值交换，right向左一步，直到指针相遇

```js
var removeElement = function (nums, val) {
  let left = 0;
  let right = nums.length - 1;
  while (left <= right) {
    if (nums[left] == val) {
      nums[left] = nums[right];
      right--;
    } else {
      left++;
    }
  }
  return left;
};

```
