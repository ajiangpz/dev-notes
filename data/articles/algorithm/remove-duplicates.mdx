# 描述

去除数组中的重复项

# 实现

```js
var removeDuplicates = function (nums) {
  if (nums.length < 1) return nums;
  let curIndex = 0;
  for (let i = 1; i < nums.length; i++) {
    if (nums[i] === nums[i - 1]) continue;
    curIndex++;
    nums[curIndex] = nums[i];
  }
  return curIndex + 1;
};
```
