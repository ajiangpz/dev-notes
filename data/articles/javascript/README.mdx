# JavaScript 面试题集

这个文件夹包含了前端开发中常见的JavaScript面试题和解答，按主题分类整理。

## 如何使用

1. 根据你需要准备的面试方向，选择相应的主题文件夹
2. 每个文件包含该主题下的常见问题、详细解答和代码示例
3. 建议先尝试自己回答问题，再查看解答进行对比

## 文件夹结构

- `basics/`: JavaScript基础概念
- `functions/`: 函数相关知识点
- `objects-prototypes/`: 对象和原型
- `async/`: 异步编程
- `es6+/`: ES6及更高版本特性
- `dom-bom/`: DOM和BOM操作
- `performance/`: 性能优化
- `design-patterns/`: 设计模式
- `testing/`: 测试相关
- `coding-challenges/`: 编码挑战
